# 鼓励储蓄罐 - 儿童积分管理系统

## 一、项目概述

### 1.1 项目名称
**鼓励储蓄罐** - 一款面向家庭的儿童行为激励管理系统。

### 1.2 项目目标
帮助家长记录和奖励孩子的良好表现，通过"鼓励换积分，积分换奖励"的正向激励机制，培养孩子的良好习惯和积极行为。

### 1.3 核心规则
- **积分获取**：每获得一朵小红花 = 2积分（可自定义）
- **积分兑换**：1积分 = 1元人民币
- **目标用户**：家长和孩子
- **数据存储**：浏览器本地 LocalStorage

## 二、页面架构与调用关系

### 2.1 页面路由结构

```
App (根组件)
├── Header (顶部导航栏，桌面端横向菜单)
├── BottomNav (底部导航栏，移动端 Tab)
├── ToastContainer (全局 Toast 提示)
├── ConfirmDialog (全局确认对话框)
└── Routes
    ├── /          → Home (首页)
    ├── /children  → Children (孩子管理)
    ├── /records   → Records (积分记录)
    └── /rewards  → RewardShop (奖励商城)
```

### 2.2 页面调用关系图

```

│                         App                                 │

          │           │           │           │
          ▼           ▼           ▼           ▼
    ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌──────────┐
    │  Home   │ │Children │ │ Records │ │RewardShop│
    └────┬────┘ └────┬────┘ └────┬────┘ └────┬─────┘
         │           │           │           │
         ▼           ▼           ▼           ▼
┌─────────────────────────────────────────────────────────────┐

│  children | records | rewards | categories                  │
└─────────────────────────────────────────────────────────────┘

页面间跳转关系：
───────────────────────────────────────────────────────────────
Home ──────→ Children (添加孩子)
    │
    ├──→ Records (查看积分记录)
    │        │
    │        └──→ AddPointModal (添加积分记录)
    │
    └──→ RewardShop (奖励商城)
             │

```

### 2.3 各页面功能定位

| 页面 | 路由 | 核心功能 | 依赖的组件 |
|------|------|---------|-----------|
| **Home** | `/` | 首页仪表盘，展示孩子列表、快捷加分入口、统计数据 | ChildCard, useStore |
| **Children** | `/children` | 孩子 CRUD 操作（添加、编辑、删除） | AddPointModal, AvatarPicker, ConfirmDialog |
| **Records** | `/records` | 积分记录列表，按日期分组，支持筛选 | PointRecordItem, AddPointModal, SegmentedControl |
| **RewardShop** | `/rewards` | 奖励商城，支持自定义奖励和兑换 | ConfirmDialog |

## 四、数据模型

```typescript
// 孩子信息
interface Child {
  id: string;
  name: string;              // 姓名
  avatar?: string;          // 头像（emoji）
  age: number;              // 年龄
  totalPoints: number;      // 当前积分余额
  createdAt: string;        // 创建时间
}

// 积分记录
interface PointRecord {
  id: string;
  childId: string;          // 关联的孩子ID
  type: 'earn' | 'spend';  // 获得/消费
  amount: number;           // 积分数量
  reason: string;           // 原因说明
  date: string;             // 记录日期
  createdAt: string;        // 创建时间
}

// 积分类别（可自定义）
interface PointCategory {
  id: string;
  name: string;             // 类别名称
  points: number;           // 积分数值
  icon: string;             // 图标（emoji）
  type: 'earn' | 'spend';  // 获得/消费
}

// 奖励（商城可兑换）
interface Reward {
  id: string;
  name: string;             // 奖励名称
  points: number;           // 所需积分
  icon?: string;            // 图标（emoji）
}
```

## 五、数据流

### 5.1 添加积分（获得积分）

```
用户操作 → Home/Records 选择孩子和类别
    ↓
AddPointModal 收集数据（childId, type, amount, reason）
    ↓
useStore.addRecord() → 更新 children 和 records 状态
    ↓
storage.setChildren() & storage.setRecords() → 持久化到 localStorage
    ↓
useToastStore.addToast() → 显示"获得 X 积分"提示
```

### 5.2 删除孩子

```
用户点击删除 → Children 页面
    ↓
openConfirm() → 显示确认对话框
    ↓
用户确认 → useStore.deleteChild()
    ↓
过滤掉该孩子的 children 和 records → 持久化
    ↓
useToastStore.addToast() → 显示"已删除"提示
    ↓
navigate('/children') → 刷新页面
```
