# Reward Jar - Children's Points Management System

## Project Overview

### Project Name
**Reward Jar** - A family-friendly children's behavior incentive management system.

### Project Goal
Help parents record and reward their children's good behavior through a positive incentive mechanism: "Earn points with good behavior, redeem points with rewards" to cultivate good habits and positive behavior in children.

### Core Rules
- **Earning Points**: 1 red flower = 2 points (customizable)
- **Redeeming Points**: 1 point = 1 RMB
- **Target Users**: Parents and children
- **Data Storage**: Browser LocalStorage

## Data Flow

### Adding Points (Earning)

```
User Action → Home/Records select child and category
    ↓
AddPointModal collects data (childId, type, amount, reason)
    ↓
useStore.addRecord() → update children and records state
    ↓
storage.setChildren() & storage.setRecords() → persist to localStorage
    ↓
useToastStore.addToast() → show "Earned X points" notification
```

### Deleting Child

```
User clicks delete → Children page
    ↓
openConfirm() → show confirmation dialog
    ↓
User confirms → useStore.deleteChild()
    ↓
Filter out the child from children and records → persist
    ↓
useToastStore.addToast() → show "Deleted" notification
    ↓
navigate('/children') → refresh page
```

# 鼓励储蓄罐 - 儿童积分管理系统

## 一、项目概述

### 1.1 项目名称
**鼓励储蓄罐** - 一款面向家庭的儿童行为激励管理系统。

### 1.2 项目目标
帮助家长记录和奖励孩子的良好表现，通过"鼓励换积分，积分换奖励"的正向激励机制，培养孩子的良好习惯和积极行为。

### 1.3 核心规则
- **积分获取**：每获得一朵小红花 = 2积分（可自定义）
- **积分兑换**：1积分 = 1元人民币
- **目标用户**：家长和孩子
- **数据存储**：浏览器本地 LocalStorage

## 五、数据流

### 5.1 添加积分（获得积分）

```
用户操作 → Home/Records 选择孩子和类别
    ↓
AddPointModal 收集数据（childId, type, amount, reason）
    ↓
useStore.addRecord() → 更新 children 和 records 状态
    ↓
storage.setChildren() & storage.setRecords() → 持久化到 localStorage
    ↓
useToastStore.addToast() → 显示"获得 X 积分"提示
```

### 5.2 删除孩子

```
用户点击删除 → Children 页面
    ↓
openConfirm() → 显示确认对话框
    ↓
用户确认 → useStore.deleteChild()
    ↓
过滤掉该孩子的 children 和 records → 持久化
    ↓
useToastStore.addToast() → 显示"已删除"提示
    ↓
navigate('/children') → 刷新页面
```
